# Enterprise ERD & Database Design

A comprehensive Entity-Relationship Diagram (ERD) and relational database schema designed for a Smart City enterprise system scenario.

## Project Overview

This project focuses on designing a well-structured database for a Smart City system, applying key database design principles including normalization, data integrity, and relational modeling.

## Contents

| File | Description |
|------|-------------|
| `Smart City Report.pdf` | Full database design report including ERD, schema definitions, normalization steps, and data dictionary |
| `Smart City Slides.pptx` | Presentation slides summarizing the design approach and key decisions |

## Key Concepts Applied

- Entity-Relationship Diagram (ERD)
- Relational Schema Design
- Normalization (1NF, 2NF, 3NF)
- Data Integrity & Constraints
- Primary & Foreign Keys
- SQL DDL

## Tools Used

- ERD Modeling Tools
- SQL
